# 添加一张有质量的截图

> 有时候，纯文字很考验作者与读者跨越时空的沟通技巧。为了降低难度、提升信息传达的精准性，平台建议您在文中适时的添加截图来辅助说明。



## 如何截图？

- 截全窗口：截取所需内容所在的完整页面。**「建议1440*768」**
- 高亮圈选：选择手电筒模式或者几何图形框出重点内容。**「线条颜色建议蓝色，贴近UCloud logo蓝色」**

- 该说就说：善用注释。**「言简意赅，对比用色」**
- 该动得动：平台支持 **.gif**和**.mp4**格式素材播放。**「建议操作说明可以使用」**
